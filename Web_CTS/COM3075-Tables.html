COM3075 - Tables Formatting with CSS

 

 

This table tag is: < table class="evenHighLight" width="400" border="1">

In the external CSS file:
/* every even line will have a lightcyan (light blue) background */
.evenHighLight tr:nth-child(even){
     background-color: lightcyan;
}

/* the 6th row down will have a yellow background */
.evenHighLight tr:nth-child(6){
     background-color: yellow;
}

/* when the mouse moves over a row it will turn the text white and the background purple */
.evenHighLight tr:hover {
     color: white;
     background-color,: purple;
}

This table tag is: < table class="oddHighLight" width="600" border="3">

In the external CSS file:
.oddHighLight {
     color: blue;
     padding: 5px;
     margin: 10px;
}
/* all of the table cells (not top row or left column) will have red text                     */
.oddHighLight td {
     color: red;
     padding: 5px;
}

/* every odd line will have a greenyellow (light bright green) background */
.evenHighLight tr:nth-child(odd){
     background-color: greenyellow;
}
