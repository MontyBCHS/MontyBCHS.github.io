If you look at the code for this page all of the CSS is inline because of the structure of a Weebly website, and I don't want you to access or change the CSS because that will affect my entire site.

Images

When choosing images there are a couple of file types to choose from:

  • .jpg - small file size, but no transparent sections, and it can blur small details due to the compression method. Good for small to medium photos.
  • .gif - less color than .jpeg, and no tranparent sections. Can be animated. Good for logos and short animations.
  • .png - larger file size than .jpg but allows for transparent sections, and has slightly better colors. Good for transparent sections for picture on picture as well has large detailed photos, but has a larger file size. .tiff - very large for raw photographs. Can be very slow to load. Generally only used for photograpers displaying their work.

<img src="file location">

Alignment

style="float:right;"

For the pictures and text in the images section you will notice that the images have the text below not beside the image. HTML tells the web browser to desiplay everything in order, one item and then the next item goes below and then the next item goes below and so on.
To get the text to flow to the left or right of the image we need to use CSS to tell the web browswer how to display the content (image and text).

style="float:left;"


To make this work put the image in your HTML with the float left or right. Then put the text you want to wrap around the image. It will make a rectangle for the image and wrap the text around the rectangle based on your previous text alignment options.
Depending on the width of your browser window the two images above they might be on either side of the screen at the same time. Adjust the width of your window to see the flow of text around each picutre change.
Some designers dictate so much CSS that they severly limit the browser's ability to adjust for different window sizes. You will notice some websites have a narrow column of text in the middle with very large left and right margins so that it will look identical on computer screens and on cell phone screens.

Float

The purpose of float is to release an element from its original position in the flow of HTML and move it to the left or right side of the screen. If there are multiple items floated to one side it depends on the browser wether they go vertically or horizontally on the side.

style="float:left;"

style="float:left;
transform: rotate(45deg);"

style="float:left;"

This has three images floated on the left side. Most browsers will arrange them horizontally.

Clear

If you want to move elements below an object floating on the left or right even if the web browser could fit part of it in. I didn't want the clear title to be beside the three images above. I used <style="clear:both;"> to force the title below the three images. You can clear:left or clear:right or clear:both (left and right). That will tell the browser to move all of the following elements below anything that is currently floating on the left or right.
